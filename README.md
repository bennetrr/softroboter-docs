Documentation for the Softroboter project we made in school.
Visit the page: https://softroboter.docs.bennetr.me/

## Development

### Dependencies

- `node@20`
- `pnpm`

### Install dependencies

```bash
pnpm install
```

### Run development server

```bash
pnpm dev
```

### Run preview server

```bash
pnpm build
pnpm preview
```
